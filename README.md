# App Móvil SIGAE - U.E.N. "José Agustín Marquiegüi"

## Descripción del Proyecto
Este repositorio contiene el desarrollo de la aplicación móvil que complementa el **Sistema Integrado de Gestión Académica Estudiantil (SIGAE)**. Esta herramienta está diseñada para digitalizar la interacción entre la institución y sus estudiantes, facilitando la consulta de notas, trámites administrativos y comunicación en tiempo real.

## Equipo de Desarrollo
- **Anfherny Barreto:** Frontend Developer (UI/UX Móvil).
- **Daniel Crespo:** Backend Developer (API REST y Lógica de Servidor).
- **Edgar Navarro:** Database Administrator (Modelado NoSQL en MongoDB).

## 🛠 Tecnologías Utilizadas
* **Backend:** PHP (API RESTful)
* **Base de Datos:** MongoDB
* **Frontend Móvil:** [Flutter/React Native]
* **Metodología:** Scrum

## 📈 Tablero de Progreso (Scrum)
Para seguir el avance del equipo en tiempo real, puedes consultar nuestro tablero de tareas en la sección **Projects** de este repositorio. Allí verás el estado de las historias de usuario bajo el marco de trabajo Scrum.

## 📁 Estructura del Repositorio
- `/api`: Endpoints PHP para comunicación con el servidor.
- `/database`: Esquemas y configuración de colecciones en MongoDB.
- `/mobile_app`: Código fuente de la aplicación móvil.
- `/docs`: Actas de reunión, cronogramas y documentación técnica.
